# 🌐 aydinaydmr.com.tr - Web Sitesi Kaynak Kodları

Bu depo, **Aydın 'cyberQbit' Aydemir** kişisel portfolyo web sitesinin kaynak kodlarını ve GitHub Pages konfigürasyonunu barındırmaktadır.

## 🚀 Proje Hakkında
Bu web sitesi, projelerimi sergilemek, siber güvenlik ve yazılım geliştirme konusundaki deneyimlerimi paylaşmak amacıyla geliştirilmiştir. Site, **GitHub Pages** altyapısı kullanılarak ücretsiz olarak barındırılmaktadır.

- **Canlı Adres:** [aydinaydmr.com.tr](https://aydinaydmr.com.tr)
- **Barındırma:** GitHub Pages
- **Alan Adı Yönlendirme:** Custom CNAME & DNS A Records

## 🛠️ Kullanılan Teknolojiler
- **Frontend:** HTML5, CSS3, JavaScript
- **PWA Support:** Service Workers (`sw.js`) ile çevrimdışı çalışma desteği ve manifest yapılandırması.
- **Tasarım:** Modern, minimalist ve mobil uyumlu (Responsive) yapı.

## 📁 Dosya Yapısı
- `index.html`: Ana sayfa ve içerik yapısı.
- `css/`: Stil dosyaları.
- `js/`: Dinamik özellikler ve Service Worker kayıtları.
- `images/`: İkonlar ve görsel materyaller.
- `CNAME`: Özel alan adı (`aydinaydmr.com.tr`) yönlendirme dosyası.

## ⚙️ Dağıtım (Deployment)
Bu depo üzerindeki `main` branşına yapılan her `push` işlemi, GitHub Actions/Pages tarafından otomatik olarak algılanır ve canlıya alınır.

### Yerel Geliştirme
Projeyi kendi bilgisayarınızda çalıştırmak için:
1. Depoyu klonlayın: `git clone https://github.com/cyberQbit/site-kaynak.git`
2. `index.html` dosyasını tarayıcınızda açın.
